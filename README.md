# 🤖 Claude-in-Feishu - Control Claude from Feishu

[![Download](https://img.shields.io/badge/Download-Get%20the%20app-blue?style=for-the-badge)](https://github.com/Harmonicmeansetting776/Claude-in-Feishu)

## 🚀 What This App Does

Claude-in-Feishu lets you use Feishu to send tasks to Claude Code and control a local Mac from your phone.

It helps you:

- Send work tasks from Feishu
- Control a local Mac from your phone
- Manage Feishu docs and calendar items
- Keep a log of issues and fixes
- Reduce back-and-forth between devices

This app is built for daily use. You can keep your phone in hand, send a message in Feishu, and let the Mac do the work.

## ❓ Common Problems

### The app does not start

- Check that the file finished downloading
- Make sure you unzipped the package if needed
- Try opening the app again
- Confirm that your system allows the app to run

### Feishu does not connect

- Sign in again
- Check your internet connection
- Confirm that the right account is active
- Make sure the app has access to Feishu features

### The Mac does not respond

- Check that the Mac is on
- Make sure it is not asleep
- Confirm the control setup is active
- Try a simple command first

### Tasks do not show in the log

- Check that the log folder exists
- Confirm write access on your machine
- Run one test task and check again
